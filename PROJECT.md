# Cyber Terminal Blog

## 项目简介

构建一个极简、高级感、终端交互式个人博客。

用户进入网站后首先看到的不是传统博客首页，而是一个模拟 Linux Terminal。

整体风格参考：

- Linear
- Vercel
- Warp Terminal
- Raycast
- Ghostty

避免：

- 黑客绿色 Matrix 风格
- 廉价赛博朋克风
- 花哨动画

目标是：

Minimal
Elegant
Professional
Technical

------

## 技术栈

Astro

TailwindCSS

TypeScript

xterm.js

Markdown

Github Pages

------

## 核心原则

内容优先

极简设计

快速加载

静态生成

无数据库

无服务端依赖

全部文章由 Markdown 自动生成
